# 行動規範

## 私たちの約束

メンバー、コントリビューター、リーダーである私たちは、年齢・体格・障害・民族・
性自認や性表現・経験の有無・国籍・容姿・人種・宗教・性的指向にかかわらず、
誰もがハラスメントのないコミュニティに参加できることを約束します。

## 基準

歓迎される行動の例:
- 他者への敬意と共感を持って接する
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを与え、また快く受け取る
- 自らの誤りに責任を持ち、影響を受けた人へ謝罪し、そこから学ぶ

許容されない行動の例:
- 性的な言動や好ましくない言い寄り
- 侮辱・中傷的なコメント、個人攻撃や政治的攻撃
- 公的・私的を問わないハラスメント
- 許可なく他者の私的情報(住所・メール等)を公開すること

## 適用と報告

不適切な行動を目撃・経験した場合は、メンテナ(`fla9ua`)へ
GitHub の通報機能、または fla9ua@gmail.com まで連絡してください。
すべての申し立ては速やかかつ公正に確認します。

本規範は [Contributor Covenant](https://www.contributor-covenant.org) v2.1 を
参考にしています。
